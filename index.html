Tentang Saya

Keterampilan

HTML

HTML adalah bahasa markup standar untuk membuat halaman web. Ini memberikan struktur dasar dari sebuah halaman web.

PHP

PHP adalah bahasa pemrograman server-side yang digunakan untuk mengembangkan aplikasi web dinamis. PHP berjalan di server dan menghasilkan HTML.

Desain Grafis

Desain grafis mencakup proses visual komunikasi menggunakan elemen-elemen seperti foto, ilustrasi, tipografi, dan layout untuk menyampaikan pesan.

Portofolio

Web HTML

Saya memiliki pengalaman dalam pengembangan web menggunakan HTML, bahasa markup fundamental untuk membangun struktur halaman web. Saya terampil dalam membuat halaman web yang responsif dan user-friendly menggunakan teknologi HTML terbaru.

PHP

Saya memiliki pengalaman luas dalam pengembangan dengan PHP, bahasa pemrograman yang kuat untuk membangun situs web dinamis dan aplikasi web. Saya terampil dalam penggunaan PHP untuk mengelola basis data, mengolah formulir, dan membangun fungsionalitas backend yang kompleks.

Desain Grafis

Saya memiliki keahlian dalam desain grafis, fokus pada pembuatan visual yang menarik dan berdampak. Pengalaman saya mencakup desain logo, materi promosi, dan ilustrasi digital. Saya menggunakan perangkat lunak seperti Adobe Illustrator dan Photoshop untuk menciptakan desain yang kreatif dan sesuai dengan kebutuhan klien.
